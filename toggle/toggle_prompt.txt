Внесите следующие изменения в HTML-документы с текстами песен и аккордами:

1. Исправьте пути к ресурсам:
   - Измените путь к иконке favicon с "../favicon.ico" на "../../favicon.ico"
   - Измените путь к CSS файлу с "../list_styles.css" на "../../CSS/song.css"
   - Измените путь к HTML файлу с "../index.html" на "../../index.html"
   - Измените путь к HTML файлу с "../index_full.html" на "../../index_full.html", сохранив при этом ID после знака "#"

2. Улучшите структуру содержимого:
   - Удалите внешний div с классом "container", оставив его содержимое
   - Переместите div с классом "header-container" на верхний уровень иерархии

3. Реорганизуйте представление текста песни:
   - Замените единый блок текста с куплетами и припевами на отдельные блоки для каждого куплета/припева
   - Для каждого куплета и припева создайте структуру:
     ```html
     <div class="verse-container">
         <div class="verse-toggle">[номер куплета или "Пр-в"]</div>
         <div class="verse-content">
             <pre>...</pre>
         </div>
     </div>
     ```
   - Для припева добавьте дополнительное оформление, обернув текст в:
     ```html
     <div class="chorus">
         <pre>...</pre>
     </div>
     ```

4. Отформатируйте текст:
   - Удалите цифры куплетов из текста самой песни (например, "1. Когда тебе..." → "Когда тебе...")
   - Добавьте отступы перед первыми строками куплетов для выравнивания с аккордами
   - Убедитесь, что все теги <pre> корректно открываются и закрываются

5. Добавьте JavaScript для интерактивности:
   - В конец документа, перед закрывающим тегом </body>, добавьте:
     ```html
     <script src="../toggle.js"></script>
     ```

6. Сохраните без изменений:
   - Мета-информацию документа (charset, viewport)
   - Заголовок страницы
   - Навигационные ссылки внизу страницы

7. Не нужно объяснять свои действия в процессе конвертации.
